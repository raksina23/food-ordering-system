# Change Request CR-001
## Title
Cart quantity not updating on duplicate item add
## Type
Bug
## Priority
High
## Status
Resolved
## Date Submitted
2026-04-29
## Description
When a user adds the same menu item to the cart more than once, the quantity does not increment. Instead, a duplicate entry is created in the cart_items table.
## Impact Analysis
- **Components Affected:** src/backend/controllers/cart.js
- **Estimated Effort:** 2 hours
- **Risk:** Low
## Resolution
Fixed using INSERT ON CONFLICT DO UPDATE. Merged via PR #8.
